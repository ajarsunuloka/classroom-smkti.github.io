Ikut Nimbrung

Jika belum pernah fork dan clone

  • fork repository (kamu akan memiliki 
    username/classroom-smkti.github.io
    )
  • clone repository hasil fork tersebut (ganti 
    username
    dengan user kamu)
    • 
  $ git clone git@github.com:username/calssroom-smkti.github.io.git

Jika sudah pernak fork dan clone

  • tambahkan upstream 
    classroom-smkti/classroom-smkti.github.io
    di clone direktori 
    • 
  $ git remote add upstream git@github.com:classroom-smkti/classroom-smkti.github.io
  $ git rebase upstream/master

  • Lakukan Perubahan yang perlu
  • Ajukan pull requests
